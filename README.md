# Market Economic Alerts

Personal open-source tool that monitors major economic events
and generates alerts for:

- EUR/USD
- XAUUSD (Gold)
- BTC/USDT

## Features
- No automated trading
- No financial advice
- No API keys required
- Fully explainable logic

## Status
Prototype (no live data, no notifications)

## Disclaimer
This tool is for educational and informational purposes only.
